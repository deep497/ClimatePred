Temperature Prediction using Machine Learning
📌 Project Overview

This project predicts minimum and maximum temperatures using historical weather data from Himachal Pradesh. The dataset was collected from a government weather data source and processed using Python. Machine learning models were trained on the cleaned dataset to estimate temperature values based on inputs such as month and year.

To make the model interactive, a web interface was built where users can enter inputs and receive predicted temperature values.

🚀 Features

Data collection from government weather dataset

Data cleaning and preprocessing using Python

Training supervised machine learning models

Prediction of minimum and maximum temperatures

Simple web interface for user input

Backend API integration for model prediction

🛠️ Technologies Used

Python

Pandas

NumPy

Scikit-learn

FastAPI / Flask

HTML

📊 Workflow

Collected historical temperature dataset of Himachal Pradesh.

Imported the dataset into Python.

Cleaned and preprocessed the data using Pandas and NumPy.

Trained supervised machine learning models for temperature prediction.

Built a backend API using FastAPI/Flask to serve the model.

Created a simple HTML web page for user input.

Connected the frontend with the backend API to display predictions.

🖥️ How to Run the Project

Clone the repository

git clone https://github.com/your-username/temperature-prediction-ml.git

Navigate to the project folder

cd temperature-prediction-ml

Install dependencies

pip install -r requirements.txt

Run the backend server

python app.py

Open the HTML page in your browser and enter inputs to get predictions.
